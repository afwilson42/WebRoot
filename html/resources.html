  1. Generic Sensor API
    A website introducing a new Generic API to use with web apps for accessing sensors & collecting data from them. Provides overivew, samples of code to use for each sensor type, and links to sites for great details of API, and of each sensor API
    Rating: 5 out of 5

  2. MDN Web Docs on Sensor APIs
    The Mozilla Developer Network web documents that cover Sensor API concents and Usage. Includes code samples for sensor detection/existance, permissions prompts on device, readings , Interfaces, Specifications, and Browser compatibility chart. Links to specifics of each sensor type are given as well. An excellent overview
    Rating: 5 out of 5

  3. Javascript APIs for sensors
    Website covers some basics of javascript APIs for 9 sensors. Has code samples, but doesnt go in depth or provide links to sites with more information.
    Rating: 4 out of 5

  4. Apple Developer site for sensors overview
    Covers the Apple Dev classes in the Core Motion framework for iPhone sensors. Very high level, and it doesnt indicate if these apply to web, Objective C, or Swift. Good for overview, but doesnt give specific aid for web app programming
    Rating: 2 out of 5
